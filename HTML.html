HTML Cheatsheet

Basic Tags

Tag Name Description
<h1> Heading Can use h1 - h6, h1 being most important.
<p> Paragraph Self-explanatory.
<br> Break Line break. No closing tag.
<hr> Horizontal Rule Thematic break. No closing tag.
<pre> Preformatted Text For example, a poem on multiple lines.
<!-> Comments Use to make a comment or for testing.
<a> Anchor (link) href="URL" or "#id" target="_blank"
<img> Image src="URL" alt="text"
<video> Video controls <source src="URL"> text </video>
<audio> Audio controls <source src="URL"> text </audio>
<ul> Unordered List Use <li> tag inside for each point.
<ol> Ordered List Use <li> tag inside for each point.

Structural Tags

Tag Name Description
<div> Division General purpose box or container.
<article> Article Use for self-contained information.
<aside> Aside For extra info aside from the main text.
<main> Main The main part of the page.
<section> Section Used to define a section of the page.
<footer> Footer Footer
<header> Header Header
<nav> Navigation Use for Nav links.
<button> Button Defines a clickable button.
<form> Form See below.
<table> Table See below.
